# Scrum poker

A vibe-coded ad-free alternative to facilitate a poker session.


### Running locally

```bash
npm install
npm run dev       # starts the app at http://localhost:3000
```

### Testing

No need to start the app first — the test server spins up automatically.

```bash
npm test              # unit + Socket.IO integration tests
npm run test:e2e      # Playwright browser tests (headless)
npm run test:e2e:headed  # same, with a visible browser window
```
